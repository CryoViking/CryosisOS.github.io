Projects

We have several projects in the works right now but they are listed as private and as such don't have a public facing link that can be viewed. This will change as soon as their is a public release. But to describe what the project include, continue reading.

List of Projects:
- Presentation Suite
This system involves a mobile app, electron app and a webserver technology. It allows the user to put together a presentation comprising of power points slides, videos and audio components all at the same time. The user can then project the presentation onto a screen and then have the option to control the presentation from any mobile device that is authorised with the presentation app on the local network.
- Remote Light Control
This system involves some electrical and electronic engineering. It involves placing an electronic switch in the pathway of a 240V circuit for a light switch circuit in a room. Connected to this electronic switch is a raspberry pi with a webserver on it that hosts an API that can then be utilised by a web browser or a mobile phone app. Allowing for remote control over the light switch. Custome built electrical-rated plastic components are also being designed and printed for use with the system.
- Anime Web Player-
This project is little bit more of a hobby project. It basically uses an electron app framework to host a video player that plays anime episodes hosted on popular free websites like gogoanime or horriblesubs. The key feature to this is a in frame autoplay feature and episode navigation controls.
